# flutter_bootstrap_widgets

A bundle of Bootstrap 3 style widgets.

## Usage

Support widgets

- CSS
  - h1/h2/h3/h4/h5/h6
  - p
  - ul/li
  - ol/li
  - table (data table)
  - input text
  - textarea
  - input checkbox
  - radio
  - select
  - multi select
  - button
- Components
  - breadcrumbs
  - alert
  - modal
  - panel
